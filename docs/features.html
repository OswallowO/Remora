FEATURES · 完整功能

完整功能清單

client v2.1.0(交易終端)+ server v1.1.0(遠端監控)。以下涵蓋兩端主要功能。

CLIENT · 交易終端

桌面客戶端 v2.1.0

🧠

OPT8c 進場引擎

三套機器學習模型並行評分(籌碼差 / 漲停反轉 / 拉高跟漲),多道前置閘門,取分數最高者進場。

📉

動態停損出場

依領漲動能分檔的動態停損,停損價硬上限保護,尾盤自動結算。

🗂️

每日自動選股

雲端管線產生特徵 → 本機 achip 籌碼計算 → 自動 stocklist,無需人工清單。

🖥️

盤中工作台

看盤 + 下單合一:報價清單、多日 K 線、五檔、下單匣、風控水位、帳戶總覽。

📈

即時 tick K 線

毫秒級跳動棒 + 量能副圖 + 進出場標記;1/5/15/60 分 + 日線。

🎫

下單匣

買賣雙 tab、價格/張數 ±微調、價別 LMT/MKT、效期 ROD/IOC/FOK。

📋

委託管理

委託中狀態顯示、右鍵改價 / 改量 / 撤單,即時同步券商回報。

💼

持倉面板

10 欄即時持倉、未實現損益、停損價、右鍵平倉 / 修改停損 / 看 K 線。

📊

績效儀表板

6 張卡片(總損益、筆勝率、日勝率、平均每日…)+ 可編輯交易紀錄表(FIFO 配對)。

⏮️

回測工作區

日期區間回放、變速回放、左終端 + 右明細表、跨日加粗、賺賠上色。

🛡️

風控與 Kill Switch

每日進場上限、連續停損熔斷、日虧上限、Esc×2 一鍵鎖單、即時水位顯示。

🔴

真單 / 模擬 徽章

正式 / 模擬模式視覺徽章,送單前驗證模式一致性,不一致拒送。

🔔

Telegram 通知

9 類分類開關(進場/出場/停損/風控/對帳/連線/啟動/一般/嚴重)+ 總開關。

🎨

主題系統

純黑主題、台股/美股漲跌色切換、等寬數字、自動深色模式。

🔌

券商連線

永豐金 Shioaji(正式 + 模擬雙模式、CA 憑證);相容 Esun。

⬇️

K 線資料採集

月曆多選日期、一鍵採集,歷史涵蓋 2021–2026。

🔄

自動更新

偵測新版 → 一鍵下載替換 → 重啟,免管理員、免解除安裝。

📖

新手教學

內建多頁互動教學,涵蓋 API 設定、回測、盤中上線全流程。

SERVER · 遠端監控

伺服器端 v1.1.0

🤖

Discord 指令

/status、/positions、/pnl、/risk、/groups、/strength、/history、/chip、/news… 完整查詢。

🔔

事件推播

進場 / 出場 / 族群觸發 / 每日盤後報告 / 風控熔斷,可逐項訂閱。

🧩

推播積木

自訂事件訊息模板、公共頻道指派、角色權限矩陣,組合通知流。

📊

Web 儀表板

magic-link 登入;主控台、自選股、警示建構器、族群、新聞、公開檢視、管理面板。

🚨

警示中心

自訂股票條件警示(漲跌幅 / 價位 / 量能 / 波動 / 連動),觸發即推播。

📅

經濟行事曆

月曆檢視 + 高影響事件提前 30 分鐘預警(美/日/台/歐)。

⚡

Cloudflared 通道

自動將本機儀表板發佈為公開 HTTPS,免 port forwarding。

🔒

安全與綁定

client ⇄ server 版本強綁定、雙 client 去重、API token 防護、稽核日誌。

⏻

電源與自癒

多種乾淨關閉方式、啟動自癒清殘留行程、PID 管理。
